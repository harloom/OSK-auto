

# OSK-auto
Tool used to help get a lot of machine information

## install
```console
pip install paramiko
```

```console
pip install colorama
```

## run

```console
osk.py -f <file>
```


## example
```console
osk.py -f list_payload/1.json -o outFile.json -l logs.json
```
